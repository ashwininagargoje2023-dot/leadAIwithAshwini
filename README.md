## How to deploy on GitHub Pages

1. Create a new GitHub repository called `leadAIwithAshwini` (or any name)
2. Upload all 3 HTML files to the root of the repo
3. Go to **Settings → Pages**
4. Under **Source** select **Deploy from a branch**
5. Select **main** branch → **/ (root)** → click **Save**
6. Your site will be live at `https://yourusername.github.io/leadAIwithAshwini/`

## Connect your custom domain

1. In GitHub Pages settings → **Custom domain** → enter `leadAIwithAshwini.com`
2. Go to your domain registrar (GoDaddy, Namecheap etc.)
3. Add these DNS records:

| Type | Name | Value |
|------|------|-------|
| A | @ | 185.199.108.153 |
| A | @ | 185.199.109.153 |
| A | @ | 185.199.110.153 |
| A | @ | 185.199.111.153 |
| CNAME | www | yourusername.github.io |

4. Wait 24–48 hours for DNS to propagate
5. Check **Enforce HTTPS** in GitHub Pages settings
